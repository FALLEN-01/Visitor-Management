About This Project

This Visitor Management System (VMS) is designed to streamline the process of managing visitors and staff within an organization. Built with Flask and modern web technologies, it offers a beautiful glassmorphic UI that's both functional and aesthetically pleasing.

The system includes comprehensive visitor tracking, staff management, reporting features, and security controls - all wrapped in an intuitive interface with both light and dark mode support.
